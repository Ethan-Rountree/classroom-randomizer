Classroom Randomizer

This app is designed to help a teacher randomly select students or groups of students. To start, enter the names of all students in your class.

Enter the names of all students separated by commas.

Virtual Hat

This tool selects a random student from the class. It will not repeat a student until everyone has been selected.

Random Groups

This tool creates random groups of students. You can choose the number of groups to divide the class into. The number can't be bigger than half the class size so that everyone has at least one partner.

Enter a number of groups to divide the class into.
